# NEURAL_OVERRIDE v9000

An interactive entry portal inspired by Cyberpunk aesthetics and advanced security systems. This project focuses on visual UX, where the code reacts dynamically to user input.

## Features
* Neural Canvas Background: A dynamic background that generates random neural pathways when typing in the input fields.
* Biometric Interface: A mock biometric readout for an advanced security feel.
* Glassmorphism UI: A modern glass-style design with Cyan Neon lighting.
* 3D Tilt Effect: Utilizes vanilla-tilt.js for interactive mouse-tracking movement.
* Fully Responsive: Optimized for all screen sizes using Bootstrap 5.

## Tech Stack
* HTML5 / CSS3 (Custom Animations and Gradients)
* JavaScript (Vanilla)
* Canvas API (To generate the neural growth effect)
* Bootstrap 5 (For structural layout)
* Vanilla-tilt.js (For the 3D tilt movement)

https://ayms-gl1tch.github.io/NEURAL_OVERRIDE-v9000/

## How It Works
When you start typing in the UID or CODE fields, the startGrowth() function is triggered. It draws neural branches from the center of the screen in random directions, creating the sensation that the system is syncing or thinking alongside the user.

## Installation
Since this project relies on core web technologies, you can run it simply by opening the index.html file in any modern browser.
